Real-world validation — structural measures

The benchmark's second source: the structural measures run on real, naturally-occurring labelled code (CodeSmellData 1.0 / 2.0 and PySmell), to check whether the findings generalise beyond the injected samples. Similarity measures need a clean twin and so do not apply to real code.
